
## Description
Nest framework TypeScript starter repository with Mongodb Prisma.


## Prerequisites

```bash
# Create .env
copy example.env to .env

# Install dependency
yarn install

#prisma
npx prisma generate
npx prisma db push
```


## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
